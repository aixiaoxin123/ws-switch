### 3. `release.ps1`（本地私有，不上传）
一键发布脚本，自动完成：构建 → 创建 GitHub Release → 上传 exe → 推送 tag。

依赖 `.env` 中的 `GITHUB_TOKEN`。用法：
```powershell
.\release.ps1        # 普通发布
.\release.ps1 -Clean # 清缓存重新构建
```

存储 GitHub Personal Access Token，供 `release.ps1` 读取，无需每次手动设置。


## 四、待处理事项

- [ ] **修复同事电脑切换后 Windsurf 不重启的 Bug**
  - 根因：新版 Windows 11 中 `wmic` 已废弃，`find_windsurf_exe()` 返回 `None`，重启被静默跳过
  - 方案：在 `windsurf_switcher.py` 中追加 PowerShell `Get-Process` 作为备用查询方式
  - 文件：`windsurf_switcher.py`（`find_windsurf_exe` 和 `start_windsurf` 两处）

- [ ] **Token 安全**：聊天中曾暴露过 Token，需到 [GitHub Settings → Tokens](https://github.com/settings/tokens) 撤销并重新生成写入 `.env`
